# BoxKit

Extensions for personal use.

* Swift 5.0
* Xcode 10.2
* CocoaPods 1.7

### CocoaPods

You can install `BoxKit` via CocoaPods,
by adding the following line to your `Podfile`:

```ruby
pod 'BoxKit', :git => 'https://github.com/utrpanic/box-kit-ios.git', :tag => 'v2.1'
```
